## 자료 구조의 경계 조건

어떠한 경계 조건에서든 경계 조건에서 문제가 생길 가능성에 대해 힝싱 고려해야 한다.

- Empty data Structure
  - 자료 구조가 비어있는 경우
- Single elements in the data structure
  - 자료 구조에 단 하나의 요소만 들어있는 경우 
- Adding / removing beginning of data structure
  - 자료구조의 첫번째 요소를 추가하거나 제거하는 경우
- Adding / removing beginning of data structure
  - 자료구조의 마지막 요소를 추가하거나 제거하는 경우
- Working in the middle
  - 자료 구조의 중간 부분을 처리하는 경우
